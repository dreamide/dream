## Packaging

Build the renderer and package the Electron app:

```sh
pnpm package
```

Artifacts are written to `release/` by `electron-builder`.

Useful variants:

```sh
pnpm package:dir
pnpm package:mac
pnpm package:win
pnpm package:linux
```
